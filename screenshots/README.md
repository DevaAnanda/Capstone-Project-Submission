# Screenshots Placeholder

Silakan tambahkan screenshots aplikasi Anda di folder ini:

1. **home_screen.png** - Screenshot halaman utama
2. **add_mood.png** - Screenshot form tambah mood
3. **detail_screen.png** - Screenshot detail mood
4. **statistics.png** - Screenshot halaman statistik
5. **camera.png** - Screenshot integrasi kamera

## Cara Mengambil Screenshot:

### Android Emulator:
- Gunakan tombol camera di sidebar emulator
- Atau tekan Ctrl + S (Windows/Linux) / Cmd + S (Mac)

### iOS Simulator:
- Cmd + S untuk save screenshot

### Device Fisik:
- Gunakan screenshot bawaan device
- Transfer ke komputer via USB

### Tips:
- Gunakan resolusi 1080x1920 atau 720x1280
- Format PNG untuk kualitas terbaik
- Pastikan tidak ada data sensitif
- Ambil screenshot dalam mode portrait
